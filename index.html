Code Stamps

A web application that allows users to 'stamp' comment interfaces on top of uploaded code.

The inspiration for this project came from the lack of applications that emphasize reading and understanding code as opposed to simply supplying the answers right away. Users can upload code, add stamps, and manipulate stamps. The link to the code can be shared, which allows multiple users to view the code and contribute to adding stamps. Stamps can be dragged, they can switch sides, change colours, and can even be injected as native comments.

Code Stamps was built with JavaScript, React, jQuery, Ruby on Rails, PostgreSQL, Bootstrap, HTML5, CSS3, and deployed with Heroku.

Try it Here.

Source: https://github.com/ryancj/codestamps

Codrlogue

A web application that showcases and connects CodeCore students with potential employers.

Codrlogue is a group project that was built over a weekend as a 'mini-hackathon' challenge. This project relied heavily on team collaboration and the use of Git/GitHub. Codrlogue provides different functionalities depending on the type of users. Students can create and personalize profiles. Employers can browse the profiles and connect with students via contact forms, email, or the private messaging system.

Codrlogue was built with Javascript, jQuery, Ajax, Git/GitHub, Ruby on Rails, PostgreSQL, Bootstrap, HTML5, CSS3 and deployed with Heroku.

Try it Here.

Source: https://github.com/jennli/codrlogue

Siren

Reactive surface that brings awareness to noise pollution.

Siren alerts users by moving and glowing to the frequencies of surrounding sounds. Subtle movement with blue light indicates an acceptable level for the human ear, and sporadic movement with red light warns users of potential harm. The interactions are caused by an Arduino board that is programmed to move the inner mechanisms and respond to the microphone. The construction of Siren involved 3D renders made with SolidWorks, Laser Cutting, and manual assembly.

Siren involved an extensive process of design(concept, materials), refinement(prototyping, critique), and fabrication(SolidWorks, laser cutting, and assembly).

More information: http://siren337.weebly.com

Code Stamps

A web application that allows users to 'stamp' comment interfaces on top of snippets of code.

The inspiration for this project came from the lack of applications that emphasize reading/understanding code as opposed to simply supplying the answers right away. Users can upload snippets of code and add stamps to them. Stamps can also be added by other users. Stamps can be dragged, they can swap sides, change colour and can even be injected as native comments.

Code Stamps was built with ReactJS, jQuery, Ruby on Rails, PostgreSQL, Bootstrap, HTML5/CSS3, Javascript deployed with Heroku.

Try it Here.

Source: https://github.com/ryancj/codestamps

Codrlogue

A web application that showcases and connects CodeCore students with potential employers.

Codrlogue is a group project built over a particular weekend as a 'mini-hackathon' challenge. This project relied extensively on team collaboration and the use of Git/Github. Codrlogue allows students to create and personalize profiles. Employers can browse these profiles and connect with students via contact forms, email, or the private messaging system.

Codrlogue was built with Javascript, jQuery, Ajax, Git, Ruby and Rails, PostgreSQL, Bootstrap, HTML5/CSS3 and deployed with Heroku.

Try it Here.

Source: https://github.com/jennli/codrlogue

Siren

Interactive surface that moves and lights up in accordance to surrounding noises

Siren is a design project that brings awareness to noise pollution. It moves and lights up according to the volume and frequency of surrounding sounds. The sculpture moves and glows blue when the surrounding noises are at an acceptable level for the human ear. If the sculpture moves in more of a fast, sporadic manner and glows red, then it means that the surrounding noises are potentially damaging or harmful to the human ear.

Siren involved SolidWorks, Arduino, Laser cutting, Interaction Design and Rapid prototyping.

More information: http://siren337.weebly.com
